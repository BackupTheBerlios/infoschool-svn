Benutzerinformationen

Über die Namen von Personen kommt man auf deren Informationsseite.
Wenn die Person es will, werden hier Daten wie Geburtsdatum oder E-Mailadresse angezeigt. Ansonsten werden alle Gruppen aufgelistet, in denen eine Person Mitglied ist.
Das Briefsymbol hinter dem Namen führt zum Erstellen einer persönlichen Nachricht.

Eigene Daten

Benutzt man den Link zur eigenen Person oben rechts, dann erscheinen die eigenen Informationen in einem Formular, in dem man alle Daten (außer der ID) ändern kann.
Die Kästchen unter "öffentlich" entscheiden, ob andere Personen im System die Daten sehen dürfen. Wenn man wissen will, wie andere Personen die eigene Seite sehen, wählt man den eigenen Namen in einer Personenliste aus.
"Messages per Mail schicken" bewirkt, dass alle Nachrichten, die man über dieses System erhält, an die angegebene E-Mailadresse geschickt und im System gleich als gelesen markiert werden.
"Vertretungsplan per Mail schicken" bewirkt, dass der Vertretungsplan von morgen um 18:00 per E-Mail verschickt wird, falls er vorhanden ist.
