Folgende Administrationsaufgaben stehen zur Verfügung:

  • MySQL-Daten ändern
  • Zufallspasswort für automatische Skripte erstellen
  • crontab einrichten

Erläuterungen:

Das Zufallspasswort muss mindestens einmal erstellt werden. Es schadet nicht, es von Zeit zu Zeit neu zu generieren. Es könnte jedoch die Systemsicherheit erhöhen.

Wenn cron zur Verfügung steht (unter Linux Standard), sollte ein crontab eingerichtet werden. Damit werden bestimmte Skripte zu bestimmten Zeiten gestartet, sodass z.B. ein Vertretungsplan um 18:00 verschickt werden kann.
Steht cron nicht zur Verfügung, muss irgendein anderer Dienst dafür genutzt werden. Unter Windows müssen die Skripte dann zum Beispiel mit dem Taskmanger gestartet werden. Um zu erfahren, was wann ausgeführt werden soll, muss man in die Datei config/crontab schauen.
Falls es nicht automatisch möglich ist Befehle auszuführen, z.B. weil kein Dienst dafür zur Verfügung steht, können die Befehle auch manuell gestartet werden.
