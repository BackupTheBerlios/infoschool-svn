Erläuterungen

1) Bei der ersten Konfiguration muss diese Option aktiviert werden. Wenn in der Datenbank schon Daten gespeichert sind, z.B. schon Benutzer registriert sind, dann werden diese alle gelöscht.
Der Benutzer admin wird dabei auch neu ohne Passwort erstellt. Wenn das nicht erwünscht ist, sollte die Option deaktiviert werden.

2) Der root-Account ist zur Administration da und hat alle Rechte. Er wird bei MySQL standardmäßig eingerichtet und hat anfangs den Benutzernamen root und kein Passwort. Allerdings sollte ihm sehr schnell ein Passwort gegeben werden, damit die Daten in der Datenbank sicher sind.

3) Der Pfad zu den Binärdateien von MySQL wird nicht benötigt, solange die Dateien in einem Systemverzeichnis sind, sodass sie von überall ausgeführt werden können.
Unter Linux ist das Standard. Unter Windows muss meist der Pfad angegeben werden. Beispiel:
C:\mysql\bin\ Er wird allerdings nur benötigt, wenn ein neuer Account für Infoschool vom Setup angelegt werden soll.
